------------

**ATLAS Plots** provides a simple interface to produce plots in `ROOT <https://root.cern/>`_ following the standard `ATLAS <https://atlas.cern/>`_ style guidelines. It uses `matplotlib <https://matplotlib.org/>`_-like syntax and idioms while still giving access to the underlying ROOT objects.
